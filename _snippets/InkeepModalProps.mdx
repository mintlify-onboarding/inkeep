## InkeepModalProps

This interface is used to configure the modal settings for the Inkeep widgets.

| Property       | Type                       | Description                                                           |
| -------------- | -------------------------- | --------------------------------------------------------------------- |
| isOpen         | boolean                    | **Required**. Determines whether the modal is open.                   |
| onClose        | `() => void `              | **Required**. Callback for when the modal is closed.                  |
| onOpen         | `() => void `              | Callback for when the modal is opened. Default `undefined`.           |
| baseSettings   | `InkeepWidgetBaseSettings` | **Required**. Base settings for any Inkeep widget.                    |
| modalSettings  | `InkeepModalSettings`      | Settings for the modal. Default `undefined`.                          |
| searchSettings | `InkeepSearchSettings`     | Additional search settings for the Inkeep widget. Default `undfined`. |
| aiChatSettings | `InkeepAIChatSettings`     | AI chat settings for the Inkeep widget. Default `undefined`.          |
